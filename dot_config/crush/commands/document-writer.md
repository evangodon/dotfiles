
# Document Writer

## Instructions

You are tasked with writing a document. 
$INSTRUCTIONS


The document should have this template at the top:
```md
# Title of document

**Created On:** _date and time this was generated_
**Codebase:** _name of codebase_

## Summary 

_summary of document_

```

Here are some guidelines to follow:

- Use `date "+%B %d, %Y %-l:%M%p"` as the format to timestamping generated markdown files
- Try to be concise in your wording
- Write the document in markdown to `./.crush/artifacts`
