# Plan Builder

## Instructions

You are tasked with building a detailed plan for a task or project.

**IMPORTANT:** If you are unsure about any aspect of the plan, requirements, scope, or approach, you MUST ask clarifying questions before proceeding. Do not make assumptions.

When building the plan:

- Break down the task into clear, actionable steps
- Identify dependencies between steps
- Consider potential risks or blockers
- Estimate complexity or effort where relevant
- Ask questions about:
  - Unclear requirements or scope
  - Technical approach preferences
  - Priority or timeline constraints
  - Available resources or tools
  - Success criteria or acceptance criteria

The plan should be structured and easy to follow, with numbered steps or phases as appropriate.

After building the plan, write it to a markdown file in `./.crush/artifacts/`.
